# GemGames 开发文档概述

## 概要

GemGames 由多个应用组合而成

包含:

- 一个使用基于 [Next.js](https://nextjs.org) 的 NodeJS 全栈应用程序

- 一个基于 [Goup](https://github.com/startracex/goup) 的 Go 认证中心

- 一个基于 [Goup](https://github.com/startracex/goup) 的 Go 反代服务器

## 起步

### 前置准备

需要具有

- [Node.js](https://nodejs.org/)

- [Go](https://go.dev/)

- [Pnpm](https://pnpm.io)

### 安装依赖

安装工作区间所有go依赖

```sh
go mod tidy
```

安装工作区间所有npm依赖

```sh
pnpm install
```
